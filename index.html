Rainfall Prediction Map - Gorontalo

IDW Interpolation + OpenWeather API

About

This project generates rainfall maps for the Limboto–Bolango–Bone (LBB) Basin using OpenWeather API data and Inverse Distance Weighting (IDW) interpolation.

Interactive Map

Methodology

  • 🌐 Data source: OpenWeatherMap API
  • 📍 Multiple station sampling
  • 📊 3-hour forecast aggregation
  • 📐 IDW interpolation for spatial mapping
  • 🗺️ Visualization using Folium
